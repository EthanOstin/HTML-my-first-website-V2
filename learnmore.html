My experience in creating my very first website !

Creating my first website was an exciting introduction to how the internet works behind the scenes. I learned that websites are built using different languages, with HTML and CSS being the most basic and important starting points. At first, everything looked complicated, but once I understood the purpose of each language, the process became much clearer and more enjoyable.


I learned that HTML (HyperText Markup Language) is used to structure a webpage. It acts like the skeleton of the website, allowing me to add headings, paragraphs, images, and links. By using simple tags such as h1 for headings and p for paragraphs, I was able to organize content and display information on the page. This helped me understand how browsers read and show website content.


I also gained a basic understanding of CSS (Cascading Style Sheets), which is used to style and design the webpage. While HTML creates the structure, CSS controls how the website looks. With CSS, I could change colors, fonts, spacing, and layout. For example, I learned how to make text blue, center headings, and add background colors. This made the website more visually appealing and professional.


Another important lesson was learning how HTML and CSS work together. HTML provides the content, and CSS improves the appearance. Even with simple knowledge, I was able to create a basic webpage that displayed text and had some styling. This gave me confidence and showed me that building websites is something that can be learned step by step.


Overall, creating my first website helped me develop problem-solving skills and introduced me to the fundamentals of web development. It showed me that with patience and practice, I can continue improving my skills and eventually build more advanced and interactive websites.
